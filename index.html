for loop

  有時候我們要做重複的事情很多很多次，這時候，就可以用迴圈的寫法減少程式碼。
比如要算5!，你可能會寫code:
ans=1*2*3*4*5
但是如果要算20!甚至100!呢?
這時就不會有人想一個個把數字打出來了吧!
這種情況，我們就可以用code:
ans=1
for i in range(1,21):
	ans=ans*(i)

這樣ans就會是20!:
2432902008176640000

for loop的寫法:
for i in range(r):
	actions

r:想要做的次數
i:i變數會從0一路增加到(r-1)，並且持續重複迴圈裡的動作，故一共會重複r次動作
actions:要做的動作

if

  如果有一個題目要求我們把1000以內是7的倍數，但不是3的倍數的數字全部印出來，
那麼只依靠for迴圈以及數學計算動作應該是無法完成這的任務，於是我們就不能只是單純的下指令叫電腦照做，
而需要讓電腦根據某些條件自己去判段要做哪些事情。
  以這題來說，我們可以用迴圈讓電腦掃過1~1000的所有數字，並讓他判斷這個數字有沒有符合是7的倍數且不是3的倍數，
如果符合，就把它印出來，code:
for i in range(1,1001):
	if i%7 == 0 and i%3 != 0:
		print(i)
