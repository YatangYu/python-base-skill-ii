Type 轉換

  先複習一下，前一篇講到，不同Type的變數基本上不會有直接互動。然而，如果我們要讓不同Type的變數可以做計算等互動時該如何操作呢?
又真的會遇到這種情形嗎?以下我們用一個例題來告訴大家。

例題:簡易平方計算機
說明:輸入一個數字給電腦，然後讓電腦輸出它的平方

解題:
  看到這個題目，相信大家第一步都會想到要先寫輸入的程式碼，如code:

a=input("請輸入要平方的數字:")

  寫完這行程式碼後，大家應該也會先執行測試看看有沒有出現什麼問題。於是，問題來了，眼尖的同學應該有發現，
我們input近來的變數a它的Type是str，而加減乘除和平方等數學計算只作用在數字上，str(字串)是無法使用的。
於是，這邊我們要想辦法讓屬於str的a轉換成int或是float等數字型的Type，我們可以用code:

a_float=float(a)

  大家可以測試，在加上這行程式碼後，我們就得到一個新的變數 a_float而它的Type是float。
接著我們就可以做我們熟悉的指數運算，加上code:

print(a_float**2)

以上是例題的講解，在python裡，我們想要改變一個變數的Type只要用那種Type的名稱加上括弧，如 int() 就可以了。

list

  我們知道了，變數的定義方式，而每一個變數都存有一個資訊，那如果遇到有好幾筆資料要記錄，我們總不可能用以往的方法創造好幾個不同名字的變數來用吧!
於是，這邊跟大家介紹一種比較複雜的Type:"list"，請大家看下面的code:

a=[0,1,2]
print(a)
a.append(3)
print(a)
print(a[0])

執行的結果會是:

[0, 1, 2]
[0, 1, 2, 3]
0

  list的定義方式如上面第一行程式碼，以"[]"將所有資料包起來，每個資料以","隔開。
  "append"可以在list的尾部加上一個新的資料，以上面例子來說，a最後會是 [0,1,2,3] 。
  a[0]是一個變數，表示了a裡面的第0個元素。你可以把list想像成是一種有系統的定義變數方式，list內的每一個元素都是一個變數。

以上是關於list的教學，只要能善用list，我們就能有系統的存下一大筆資料了。

判別

  寫程式時可以提供條件，讓電腦判斷，如果條件成立就執行某動作。
比如我想做一個簡單的互動，讓我們可以輸入一個數字給電腦判別是不是7的倍數，如果是，就印個"yes"出來，就可以寫code:
a=int(input("請輸入要判別的數字:"))
if a%7 == 0:
	print("yes")

這樣我們就可以在執行視窗裡做到輸入數字，而當它除以7的餘數是零(也就是整除，如7777,8904)，電腦就會執行print("yes")程式碼，而在執行視窗印出"yes"。

if的寫法:
if judge:
	actions

judge:判斷的條件
actions:條件成立時要做的事

else & elif

  我們也可以設定當條件不成立時要執行的事情(else)，也就是可以藉由一個條件判斷，把事情一分為二。
以前面的例子來說，我們想知道輸入的數字是不是7的倍數，但依照原本的寫法，電腦只會在成立時告訴我們數字是7的倍數，
不成立時，不會有動靜。若要讓這個程式完整一點我們可以設計在數字不是7的倍數時顯示"no"，code:

a=int(input("請輸入要判別的數字:"))
if a%7 == 0:
	print("yes")
else:
	print("no")


  在原先的程式碼後面再加入else以及要做的動作，就可以讓程式再條件未成立時執行else內的程式碼。另外如果有些情況需要把事件做更多分岔，
我們就可以用elif(else if)的方式，比如下面的code:

a=int(input("請輸入要判別的數字:"))
if a>0:
	print(a, "is positive")
elif a<0:
	print(a, "is negative")
else:
	print(a, "is zero")


  這裡的程式碼讓我們可以輸入一個數字，而電腦會告訴我們數字是正的、負的、或是0。
另外，請大家注意if跟elif的判別條件不一定要將事件一分為二，他們是可以有交集的，請看以下code:

a=int(input("請輸入要判別的數字:"))
if a%7 == 0:
	print(a, "is a multiple of 7")
elif a%3 == 0:
	print(a, "is a multiple of 3")
else:
	print(a, "is nothing")


  這個程式，可以讓電腦告訴我們輸入的數字是7的倍數、3的倍數還是都不是。妳可以試試看輸入21讓電腦判斷，結果會是7的倍數；
但我們知道21也是3的倍數，然而卻不會顯示出來，是因為電腦在判別到if裡的條件成立後，就不會再判別elif的條件了。

loop(迴圈)

for

  有時候我們要做重複的事情很多很多次，這時候，就可以用迴圈的寫法減少程式碼。比如要算5!，你可能會寫code:

ans=1*2*3*4*5

但是如果要算20!甚至100!呢?這時就不會有人想一個個把數字打出來了吧!我們可以用code:

ans=1
for i in range(1,21):
	ans=ans*i


這樣ans就會是20!:
2432902008176640000

for loop的寫法:
for i in range(r):
	actions

r:想要做的次數
i:i變數會從0一路增加到(r-1)，並且持續重複迴圈裡的動作，故一共會重複r次動作
actions:要做的動作
  **range也可以設定想做的區間，如range(1,20)就會使i從1跑到(20-1)

while

  除了for loop以外，還有一個while loop，while是結合for與if而成的迴圈，請看code:

b=10
while b>=0:
	b=b-1
print("b=", b)

執行這段程式碼，會看到:

b= -1

由範例我們可以知道，b=b-1這行程式碼被執行了11次，b才會從原本的10變成-1，也因-1已經不符合b>=0的條件而脫離了這個迴圈。

indentation(縮排)

  不知道大家有沒有注意到，前面的if,else,for等在設定完條件後會有一個":"，而對於符合條件下我們要執行的程式碼，
是把它放在下一行且開頭要加"tab"，這就是縮排。寫python時，要把符合條件要執行的程式碼都加一個"tab"，代表這些程式碼是被包在這個迴圈或判別下。
  (常常程式出bug都是因為沒縮排的問題)
