Type 轉換

  先複習一下，前一篇講道，不同Type的變數基本上不會有直接互動。然而，如果我們要讓不同Type的變數可以做計算等互動時該如何操作呢?
又真的會遇到這種情形嗎?以下我們用一個例題來告訴大家。

例題:簡易平方計算機
說明:輸入一個數字給電腦，然後讓電腦輸出它的平方

解題:
  看到這個題目，相信大家第一步都會想到要先寫輸入的程式碼，如code:

a=input("請輸入要平方的數字:")

  寫完這行程式碼後，大家應該也會先執行測試看看有沒有出現什麼問題。於是，問題來了，眼尖的同學應該有發現，
我們input近來的變數a它的Type是str，而加減乘除跟平方等數學計算只作用在數字上，str(字串)是無法使用的。
於是，這邊我們要想辦法讓屬於str的a轉換成int或是float等數字型的Type，我們可以用code:

a_float=float(a)

  大家可以測試，在加上這行程式碼後，我們就得到一個新的變數 a_float而它的Type是float。
接著我們就可以做我們熟悉的指數運算，加上code:

print(a_float**2)

以上是例題的講解，在python裡，我們想要改變一個變數的Type只要用那種Type的名稱加上括弧(如int())就可以了。


判別


  寫程式時可以提供條件，讓電腦判斷，如果條件成立就執行某動作。
比如我想做一個簡單的互動，讓我們可以輸入一個數字給電腦判別是不是7的倍數，如果是，就印個"yes"出來，就可以寫code:
a=int(input("請輸入要判別的數字:"))
if a%7 == 0:
	print("yes")

這樣我們就可以在執行視窗裡做到輸入數字，而當它除以7的餘數是零(也就是整除，如7777,8904)，電腦就會執行print("yes")程式碼，而在執行視窗印出"yes"。

if的寫法:
if judge:
	actions

judge:判斷的條件
actions:條件成立時要做的事


else & elif


  除了利用if的方式判斷要不要做某件事，我們也可以設定當條件不成立時要執行的事情(else)，也就是可以藉由一個條件判斷，
把事情一分為二。以前面的例子來說，我們想知道輸入的數字是不是7的倍數，但依照原本的寫法，電腦只會在成立時告訴我們數字是7的倍數，
不成立時，是不會有動靜的。而如果要讓這個程式完整一點我們可以設計在數字不是7的倍數時顯示"no"，code:

a=int(input("請輸入要判別的數字:"))
if a%7 == 0:
	print("yes")
else:
	print("no")


  在原先的程式碼後面再加入else以及要做的動作，就可以讓程式再條件未成立時執行else內的程式碼。另外如果有些情況需要把事件做更多分岔，
我們就可以用elif(else if)的方式，比如下面的code:

a=int(input("請輸入要判別的數字:"))
if a>0:
	print(a, " is positive")
elif a<0:
	print(a, " is negative")
else:
	print(a, " is zero")


  這裡的程式碼讓我們可以輸入一個數字，而電腦會告訴我們數字是正的、負的、或是0。另外，請大家注意if跟elif的判別條件不一定要將事件一分為二，
他們是可以有交集的，請看以下code:

a=int(input("請輸入要判別的數字:"))
if a%7 == 0:
	print(a, " is a multiple of 7")
elif a%3 == 0:
	print(a, " is a multiple of 3")
else:
	print(a, " is nothing")






for loop


  有時候我們要做重複的事情很多很多次，這時候，就可以用迴圈的寫法減少程式碼。
比如要算5!，你可能會寫code:
ans=1*2*3*4*5
但是如果要算20!甚至100!呢?
這時就不會有人想一個個把數字打出來了吧!
這種情況，我們就可以用code:
ans=1
for i in range(1,21):
	ans=ans*(i)

這樣ans就會是20!:
2432902008176640000

for loop的寫法:
for i in range(r):
	actions

r:想要做的次數
i:i變數會從0一路增加到(r-1)，並且持續重複迴圈裡的動作，故一共會重複r次動作
actions:要做的動作
